# การอัปเดตกฎภาษาสำหรับ AI Agent (Global)

ผมได้ดำเนินการแก้ไขและปรับปรุงกฎภาษา (Language Policy) ระดับ Global เพื่อให้การทำงานในทุกโปรเจกต์มีความสม่ำเสมอและตรงตามความต้องการของคุณครับ

## รายงานการเปลี่ยนแปลง

### 1. การแก้ไขไฟล์กฎ Global
ผมได้ทำการแก้ไขไฟล์ `/Users/oatrice/.gemini/GEMINI.md` ซึ่งเป็นจุดเก็บกฎหลักที่ส่งผลต่อพฤติกรรมของ AI Agent ในระบบของคุณ

> [!IMPORTANT]
> กฎใหม่ที่เพิ่มเข้าไปมีดังนี้:
> - **Chat Responses**: ต้องตอบเป็นภาษาไทยเสมอ (ยังคงเดิม)
> - **Git Operations**: ข้อความ Git commit และคำอธิบาย Pull Request **ต้องเป็นภาษาอังกฤษเท่านั้น**
> - **Internal Documentation**: ไฟล์ `task.md`, `implementation_plan.md`, และ `walkthrough.md` **ต้องเป็นภาษาไทย**
> - **Standard Project Documentation**: ไฟล์ `README.md`, `CHANGELOG.md`, และ `ROADMAP.md` **ต้องเป็นภาษาอังกฤษเท่านั้น**

### 2. การจัดโครงสร้างเนื้อหาใหม่
ผมได้จัดรูปแบบเนื้อหาในไฟล์ `GEMINI.md` ให้เป็นระเบียบมากขึ้น โดยแยกส่วน "Language Policy" ออกมาให้ชัดเจนและเพิ่ม Emoji เพื่อให้อ่านง่ายขึ้น
